1) Download and Install Mongodb Database from https://www.mongodb.com/try/download/community
2) Download and Install Nodejs from https://nodejs.org/en/download/prebuilt-installer
3) open Backend Folder and type "npm install" in Terminal after completing type "npm start" in terminal
4) open Frontend Folder and type "npm install" in Terminal after completing type "npm start" in terminal
5) Open .env file and add email (only gmail supported) and add app password(this is not your email password, to get app password watch tutorial)
6) Now add some rooms by navigating to http://localhost:5173/addroom.
